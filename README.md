# Dictation Master (听力大师) 🎧

**Dictation Master** 是一款专为英语听力练习设计的本地优先（Local-first）Web 应用程序。它允许你通过加载本地视频和字幕文件，进行高效的“精听”和“听写”练习。

无需将视频上传到任何服务器，所有操作都在你的浏览器中本地完成，保护隐私且响应极快。

---

## ✨ 核心功能

- **本地视频支持**：直接加载本地 `.mp4` 或 `.webm` 格式视频。
- **双语字幕对齐**：支持同时加载中英两个字幕文件（`.srt` 或 `.vtt`），实现中英对照。
- **实时听写反馈**：输入你听到的内容，程序会实时比对并高亮错误内容，帮助你精准定位听力盲点。
- **单句循环播放**：自动在当前句子的时间段内进行循环，直到你听清为止。
- **倍速播放控制**：支持 0.5x 到 1.25x 的多级速度调节，适合不同难度的练习。
- **完全快捷键操作**：深度优化录入逻辑，无需频繁点击鼠标，让你的注意力集中在听力本身。

## 🚀 快速开始

### 前置条件
- 已安装 [Node.js](https://nodejs.org/)。

### 安装与运行
1. **克隆或下载本仓库**到本地。
2. **安装依赖**：
   ```bash
   npm install
   ```
3. **启动应用**：
   ```bash
   npm run dev
   ```
4. **访问应用**：在浏览器中打开命令行输出的地址（通常是 `http://localhost:5173`）。

---

## 📖 使用教程

1. **导入项目**：
   - 启动应用后，在上传页面选择你的 **视频文件**（支持主流浏览器兼容格式）。
   - 选择对应的 **英文格式字幕**（`.srt` 或 `.vtt`）。
   - （可选）选择 **中文格式字幕** 以便在完成后查看对照翻译。
   - 点击 **Start Practice** 进入练习界面。

2. **听写流程**：
   - 视频会自动定位并循环播放当前句子。
   - 在右侧输入框中输入你听到的英文。
   - 按下 **`Enter`** 键检查结果：
     - 如果输入存在错误，程序会高亮显示错误单词，并自动选中输入框中第一个出错的位置，方便你立即修改。
     - 如果输入完全正确，程序会显示翻译，此时再次按下 **`Enter`** 即可跳转到下一句。

3. **进阶模式**：
   - 如果实在听不出来，可以点击 **"Give Up"** 查看标准答案。
   - 使用侧边栏的进度列表，可以随时回顾已完成或跳过的句子。

## ⌨️ 快捷键指南

| 快捷键                                    | 功能描述                            |
| :---------------------------------------- | :---------------------------------- |
| **`` ` `` (反引号)** / **`Ctrl + Space`** | **播放 / 暂停** 视频                |
| **`Enter`**                               | **检查** 听写结果 / **跳转** 下一句 |
| **`Ctrl + R`**                            | **重听** 当前句子                   |
| **`Ctrl + →`**                            | 跳转到 **下一句**                   |
| **`Ctrl + ←`**                            | 返回 **上一句**                     |

## 🔒 隐私与安全

Dictation Master 是一款 **纯本地运行** 的静态应用。你的视频文件、字幕内容以及所有录入内容仅在你的浏览器内存中处理，**不会上传到任何服务器**。你可以放心用于私人学习资料的练习。
